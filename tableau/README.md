This folder contains the Tableau Packaged Workbook (.twbx) for the Human Activity Recognition project.

Dashboard includes:

Activity-Level Mean vs Standard Deviation
Bar chart comparing average sensor values and variability.

Most Stable Activities (Least Variation Bar Chart)
Shows activities with the lowest variation (std of std Y).

Sensor Correlation: Mean vs Standard Deviation
Scatter plot showing the relationship between mean and standard deviation.

All charts are combined into an interactive dashboard for easy comparison and exploration.
